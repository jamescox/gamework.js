Mathematical utilities

Constants

pi : number

The mathematical constant π defined as 3.141592653589793.

tau : number

The 'new' circle constant τ (tau) defined as 2π. See: tauday.com for more information.

Trigonometry

angleunits : string

Represents the unit of measure of angles as used in the trigonomic functions. Possible values are:

  • 'rad' for radians or,
  • 'deg' to use degrees (this is the default).
example:
// Set the units to radians.
angleunits = 'rad';
assert(cos(pi) === -1.0);

// Set the units to degrees.
angleunits = 'deg';
assert(cos(90) === -1.0);

sin : function (angle : number) → number

Returns the trigonometric sine of an angle.

parameters:

angle - the angle, in the currently selected units, see: angleunits.

returns:

The sine of the argument.

cos : function (angle : number) → number

Returns the trigonometric cosine of an angle.

parameters:

angle - the angle, in the currently selected units, see: angleunits.

returns:

The cosine of the argument.

tan : function (angle : number) → number

Returns the trigonometric tangent of an angle.

parameters:

angle - the angle, in the currently selected units, see: angleunits.

returns:

The tangent of the argument.
